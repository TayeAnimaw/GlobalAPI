# GlobalAPI

**GlobalAPI** is a professional Golang REST API project designed for scalable and maintainable backend services. It simulates real-world enterprise API development with modular architecture, controllers, services, and routes.

## Features
- RESTful API endpoints
- Modular folder structure (controllers, services, models, routes)
- Easy to extend for new services
- Clear and professional code organization
- Ready for database integration and authentication

## Tech Stack
- Golang
- net/http for server
- Optional: GORM for database
- Optional: JWT for authentication

## Getting Started
```bash
git clone https://github.com/TayeAnimaw/GlobalAPI.git
cd global-api
go mod tidy
go run main.go
